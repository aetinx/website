Thank you for downloading the HumanSystem sound scheme/sample pack!

This project was born from my love for older operating systems and the little user experience details - like startup sounds and alerts, that gave them personality. I want to bring that feeling back, both as a working Windows sound scheme and as a collection of samples you can use in your own music. I hope you enjoy the sounds as much as I enjoyed creating them for you!

If you would like to install the sound scheme on Windows, please download the installer here:

https://alfaxysm.net/HumanSystemSounds.zip

------------------------------------------

On newer versions of Windows, you may see a SmartScreen warning saying it prevented the unrecognized application from starting - this is normal behavior for new applications without a digital signature.

The installer is safe and free of malware.
If you trust the program, simply click: "More info" -> "Run anyway".

Alternatively, if you prefer not to run the installer, you can apply the sounds manually from the  
"Sounds (samples)" directory.

------------------------------------------

*** NOTE ***
This sound scheme is designed primarily for Windows 95/98/Me/NT4/2000 systems, though it will also install on newer versions.
Sound events introduced in later Windows releases (such as hardware insertion/removal) are not yet covered. Future updates will expand event coverage!
